What FAST does
Three Validation Modes
Basic — self-review against built-in rules. Specific — validate against your business assertions. Benchmark — compare two PDF versions side-by-side.
Visual Diff Engine
Page-by-page pixel comparison with annotated side-by-side panels. Detects moved fields, text changes, signature regions, and font differences automatically.
Multi-Model AI Analysis
Choose between Claude, Gemini, or GPT-4o as the analysis engine. Findings are categorised by type (spelling, format, value mismatch, compliance) and severity.
Review & Approval Workflow
Assign findings to reviewers, confirm defects, mark false positives. Formal approval gate with named sign-off before a run is marked production-ready.
False Positive Learning
Mark an observation as a false positive once — FAST learns the pattern and suppresses the same noise automatically on every future run. No repeated manual dismissal.
Jira Integration
Log confirmed defects directly to Jira Cloud with one click from the Review Findings page. Ticket keys persist across re-runs so history is never lost.
Trend Analytics
Per-test-case run history charts filtered by Pass / Fail / In Review. Spot regressions, track quality improvement, and see the effect of process changes over time.
Evidence Bundle & Audit Trail
Every action — review, resolution, approval — is timestamped in the Audit Log. Download the full evidence bundle (annotated PDF + diff images + report) for compliance submissions.
